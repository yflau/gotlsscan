# goTLSScan

`gotlsscan` scans a tls/https server, and reports compatible Versions and Ciphers.

```
Usage of gotlsscan:
  -ca string
        specify ca file path in pem foramt(default to system cert pool)
  -host string
        host
  -insecure
        skip certificate verification
  -port string
        port (default "443")
```
